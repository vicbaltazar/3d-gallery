# 3D Gallery

Galeria 3D simples feita com **Vite + TypeScript + Tailwind CSS**.

As imagens ficam organizadas em um carrossel 3D que gira automaticamente, inspirado em um exemplo clássico do CodePen, mas adaptado para um projeto moderno com TypeScript.

## Demo

Acesse o projeto online:  
https://threed-gallery-bwok.onrender.com

## Tecnologias

- Vite
- TypeScript
- Tailwind CSS
- HTML/CSS

## Como rodar o projeto

```bash
# instalar dependências
npm install

# rodar em modo desenvolvimento
npm run dev
